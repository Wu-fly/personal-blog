# 文章审核预览功能优化说明

## 修改内容

**文件**: `blog-admin/src/views/ArticleReview.vue`

### 1. 添加 Markdown 转 HTML 功能

添加了 `convertMarkdownToHtml` 函数，将 Markdown 格式的文章内容转换为 HTML：

```javascript
const convertMarkdownToHtml = (markdown) => {
  // 支持的 Markdown 语法：
  // - 图片：![alt](url)
  // - 链接：[text](url)
  // - 标题：# ## ###
  // - 粗体：**text**
  // - 斜体：*text*
  // - 代码块：```code```
  // - 行内代码：`code`
  // - 引用：> text
  // - 列表：* item 或 - item
  // - 段落自动识别
}
```

**特点**:
- 自动检测内容格式（HTML 或 Markdown）
- 如果已经是 HTML 格式，直接使用
- 如果是 Markdown 格式，自动转换为 HTML
- 支持常用的 Markdown 语法

### 2. 添加计算属性处理内容

添加了 `processedContent` 计算属性，智能处理文章内容：

```javascript
const processedContent = computed(() => {
  if (!currentArticle.value || !currentArticle.value.content) return ''
  
  const content = currentArticle.value.content
  // 检查是否已经是 HTML 格式
  if (content.includes('<img') || content.includes('<h1') || content.includes('<h2')) {
    return content
  }
  
  // 是 Markdown 格式，需要转换
  return convertMarkdownToHtml(content)
})
```

### 3. 添加封面图片显示

在预览对话框中添加了封面图片显示区域：

```vue
<!-- 封面图片 -->
<div v-if="currentArticle.coverImage" class="cover-image">
  <img :src="currentArticle.coverImage" alt="封面图片" />
</div>
```

**特点**:
- 只有当文章有封面图片时才显示
- 图片居中显示
- 最大宽度100%，最大高度400px
- 圆角和阴影效果，美观大方

### 4. 优化文章内容中的图片显示

添加了深度选择器样式，确保文章内容中的图片能正确显示：

```css
.article-content :deep(img) {
  max-width: 100%;
  height: auto;
  border-radius: 8px;
  margin: 10px 0;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  display: block;
}
```

**特点**:
- 图片自适应宽度，不会超出容器
- 保持图片原始宽高比
- 圆角和阴影效果
- 上下留有间距

### 5. 增强文章内容样式

为文章内容添加了完整的样式支持：

- **标题样式**: h1, h2, h3 有不同的字体大小和颜色
- **段落样式**: 合理的行距和间距
- **列表样式**: ul, ol 有适当的缩进
- **引用样式**: blockquote 有左侧边框和斜体
- **代码样式**: 行内代码和代码块有不同的背景色
- **链接样式**: 带下划线和悬停效果
- **表格样式**: 边框和背景色

### 6. 添加付费标识

在文章元信息中添加了付费文章标识：

```vue
<span v-if="currentArticle.isPaid" class="paid-tag">付费文章 ¥{{ currentArticle.price }}</span>
```

## 支持的 Markdown 语法

1. **图片**: `![图片描述](图片URL)` → `<img src="图片URL" alt="图片描述" />`
2. **链接**: `[链接文本](URL)` → `<a href="URL" target="_blank">链接文本</a>`
3. **标题**: `# 标题` → `<h1>标题</h1>`（支持 #, ##, ###）
4. **粗体**: `**文本**` → `<strong>文本</strong>`
5. **斜体**: `*文本*` → `<em>文本</em>`
6. **代码块**: ` ```代码``` ` → `<pre><code>代码</code></pre>`
7. **行内代码**: `` `代码` `` → `<code>代码</code>`
8. **引用**: `> 引用文本` → `<blockquote>引用文本</blockquote>`
9. **列表**: `* 项目` 或 `- 项目` → `<ul><li>项目</li></ul>`
10. **段落**: 自动识别并包裹在 `<p>` 标签中

## 优势

1. **完整信息展示**: 管理员可以看到文章的封面图片，更好地判断文章质量
2. **图片正确渲染**: 文章内容中的 Markdown 图片语法会自动转换为 HTML img 标签
3. **智能格式检测**: 自动识别内容是 HTML 还是 Markdown 格式
4. **美观的样式**: 统一的样式设计，提升审核体验
5. **响应式设计**: 图片自适应容器宽度，不会溢出
6. **付费标识**: 清晰标识付费文章，方便管理员区分
7. **兼容性好**: 同时支持 HTML 和 Markdown 两种格式

## 注意事项

1. 图片URL必须是可访问的完整URL
2. 如果图片加载失败，浏览器会显示默认的图片加载失败图标
3. 封面图片只在存在时才显示，避免空白区域
4. 使用了Vue 3的深度选择器 `:deep()` 来穿透scoped样式
5. Markdown 转换是在前端进行的，不会影响数据库中的原始内容
6. 如果内容已经是 HTML 格式，会直接使用，不会重复转换
